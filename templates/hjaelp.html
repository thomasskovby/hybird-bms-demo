Om Hybirds API

Hybird tilbyder et fuldt API på alle kundekonti, hvor en stor del af måledata er eksponeret. Data kan hentes via API efter at man har oprettet en API Token under sin bruger.

Man kan oprette specifikke brugere til bestemte formål. For eksempel hvis man vil lave en integration til et CTS-system, kan man med fordel oprette en bruger med navnet API-CTS og herinde udtage en token.

Brugerroller:
Admin – fuld adgang til alt, inkl. brugeradministration
Operator – adgang til alt undtagen brugeradministration
Observer – kun læseadgang (anbefalet til API-integrationer)

8

Tilføj din API-konto

Klik på + Tilføj konto. Indsæt:

  1. Kontonavn: Et genkendeligt navn, f.eks. TS Sommerhus
  2. Email: API-brugerens email, f.eks. api-bms@hybird.energy
  3. API Token: Den fulde token-streng fra trin 6

Klik Tilføj og derefter Hent alle for at hente data fra Hybird-platformen.

Resultat: Dashboardet viser nu dine sites, breaker sets, forbrug og alarmer hentet direkte fra Hybirds API.

Del 4: Direkte API-brug (valgfrit)

9

Autentifikation

Hybirds API bruger Basic Authentication. Når du har en API Token fra platformen, kan du bruge den direkte i HTTP-kald.

API-dokumentation: Kontakt Hybird for fuld API-dokumentation med alle tilgængelige endpoints.
